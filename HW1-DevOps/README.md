### Requirements
- Website: https://github.com/CSC-DevOps/Course/blob/master/HW/HW1-V.md
- [PDF](HW0_requirements.pdf)

### How to Run:
- run 'npm install'
- run 'npm link'
- To create VM, run 'v up'
- To ssh into VM, run 'v ssh'
- run 'cd App'
- run 'node main.js start 9000'
